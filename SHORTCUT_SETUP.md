# PayGuard — iOSショートカット設定手順書

## 概要

PayGuardをiOSショートカットと連携することで、PayPayやApple Payなどの決済アプリを開く前に自動的に「記録フォーム」が表示されます。

---

## Step 1: PayGuardをホーム画面に追加する

PayGuardはPWA（Webアプリ）です。まずホーム画面に追加してください。

1. **SafariでPayGuardを開く**（ChromeではなくSafariを使用してください）
2. 画面下部の **共有ボタン（↑）** をタップ
3. **「ホーム画面に追加」** をタップ
4. 名前を「PayGuard」のままにして **「追加」** をタップ

> 通知機能を使う場合は、ホーム画面のPayGuardアイコンから開いてください。

---

## Step 2: iOSショートカットを作成する

### PayPayショートカットを作る

1. iPhoneで **「ショートカット」** アプリを開く
2. 右上の **「＋」** をタップして新規作成
3. **「アクションを追加」** → **「Webページを開く」** を選択
4. URLに以下を入力:

```
https://[あなたのPayGuard URL]/?record=1&app=paypay
```

> 💡 PayGuardのURLは、アプリ内の **設定 → ショートカット連携 → URLをコピー** から取得できます

5. ショートカット名を **「💴 PayPayで支払う」** に変更
6. アイコンと色をお好みで設定
7. **「完了」** をタップ

### Apple Payショートカットを作る

同じ手順で、URLを以下に変更:

```
https://[あなたのPayGuard URL]/?record=1&app=applepay
```

ショートカット名を **「🍎 Apple Payで支払う」** に設定

---

## Step 3: ショートカットをホーム画面に追加する

1. 作成したショートカットを長押し → **「詳細」**
2. **「ホーム画面に追加」** をタップ
3. アイコンがホーム画面に表示されたら完了！

---

## 使い方

### 通常の決済フロー

```
❌ 以前:
決済アプリを直接タップ → 無意識に支払い

✅ PayGuard導入後:
ホーム画面の「PayPayで支払う」をタップ
  ↓
PayGuardが開く（金額・カテゴリ・理由を入力）
  ↓
「記録して決済へ →」をタップ
  ↓
PayPayが自動で開く 💳
```

---

## URLパラメータ一覧

| パラメータ | 値 | 説明 |
|----------|---|------|
| `record` | `1` | 記録モードで開く |
| `app` | `paypay` / `applepay` / `linepay` / `id` / `suica` | 決済後に開くアプリ |

例:
- `?record=1&app=paypay` → PayPay用
- `?record=1&app=applepay` → Apple Pay用
- `?record=1&app=linepay` → LINE Pay用

---

## 各決済アプリのURLスキーム

| アプリ | URLスキーム |
|-------|-----------|
| PayPay | `paypay://` |
| Apple Pay | `shoebox://` |
| LINE Pay | `linepay://` |
| Suica（モバイルSuica） | `suica://` |

---

## トラブルシューティング

### 決済アプリが開かない
- URLスキームは決済アプリがインストールされている場合のみ動作します
- 「その他」を選んだ場合は手動でアプリを開いてください

### 通知が来ない
- ホーム画面のPayGuardアイコンから一度起動してください
- 設定 → 通知 → 「有効にする」をタップ
- iOS設定 → PayGuard → 通知を許可してください

### ショートカットからSafariが開く
- ショートカットの設定で「Safariではなく別のブラウザで開く」を変更できます
- または「Webページを開く」の代わりに「URLを開く」アクションを試してください

---

## ヒント

- 決済前の「一呼吸」がポイントです。フォームを入力する数秒が衝動買いを防ぎます
- 理由を具体的に書くほど、振り返りが役立ちます
- 週次・月次で分析画面を確認する習慣をつけましょう
